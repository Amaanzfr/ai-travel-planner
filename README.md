# ✈️ AI Travel Planner (Full Stack Web Application)

A full-stack web application that generates personalized, day-by-day travel itineraries based on user preferences such as destination, budget, interests, and trip duration.

This project demonstrates full-stack development using React for the frontend and FastAPI for the backend, along with RESTful API communication between the two layers.

---

## 🎯 Project Overview

The AI Travel Planner allows users to quickly generate structured travel plans without manually researching destinations. The application simulates an AI-powered experience by dynamically generating itineraries based on user input.

This project was built to demonstrate real-world full-stack architecture and API design principles.

---

## 🚀 Features

- Dynamic travel itinerary generation based on user inputs
- Supports destination, budget level, interests, and trip duration
- Clean full-stack separation (frontend + backend)
- REST API communication using JSON payloads
- Structured day-by-day itinerary format (Morning / Afternoon / Evening)
- Scalable backend design ready for AI model integration
